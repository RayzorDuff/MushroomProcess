üß≠ FUNCTIONAL INTENT

The Package Freeze Dried interface supports the final stage of the mushroom production lifecycle:

Tracks and converts trays of freeze-dried mushrooms into packaged retail units.

Records yield metrics between wet and dried weights.

Provides button-driven automation to mark actions as completed for either Freeze Dry or Package stages.

Filters ensure only active trays (fresh or freezing) appear to prevent double handling.

This view is functionally tied to the earlier Harvest and Print Queue interfaces, acting as the packaging endpoint.
