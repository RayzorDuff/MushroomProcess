üß≠ FUNCTIONAL INTENT

The Lots by Status interface provides a kanban-style pipeline visualization for monitoring all lots in production.
Each column represents a production or storage state, and cards within show contextual metadata like strain, item category, and vendor.
Users can:

Drag records between columns to update status directly.

Filter by item_id or strain using the dropdown.

Search or sort dynamically.

This interface acts as a high-level operational dashboard across all categories (grain, LC, plates, fruiting blocks, etc.), making it ideal for supervisors to see progress by stage.
