üß≠ FUNCTIONAL INTENT

The Landing interface acts as the primary operational dashboard for production tracking:

Displays all lots grouped by item type (e.g., grain, LC flask, substrate, plate).

Each lot record links to all downstream process pages through the embedded ‚ÄúGo To‚Äù buttons.

Provides visibility into recipe, inoculation, and input metadata without enabling editing.

Essentially this serves as the ‚Äúrouter‚Äù or ‚Äúhub‚Äù of the MushroomProcess production workflow UI.
