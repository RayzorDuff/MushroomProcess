üß≠ FUNCTIONAL INTENT

The Print Queue interface visualizes active and completed label jobs across all workflow categories.
Each Kanban column represents a print context (e.g., Grain Inoculated vs. LC Flask Inoculated).
Operators can:

Monitor pending labels grouped by stage.

Quickly print by selecting cards.

Review label details (e.g., company, subtitle, inoculation/sterilization dates, use-by).

Add manual print records through the form.

This interface functions as a central production labeling dashboard within MushroomProcess.
