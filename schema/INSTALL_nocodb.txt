Ensure prerequisites:

npm install axios

Ensure the directory structure:

schema/
  create_nocodb_from_schema.js  
  export/
    _schema.json                 
    lots.json
    events.json
    ...

In PowerShell, set env vars and run:

cd path\to\MushroomProcess\schema

$env:NOCODB_URL      = "http://localhost:8080"      # or your cloud URL
$env:NOCODB_BASE_ID  = "p_xxxxx"                    # from NocoDB UI
$env:NOCODB_API_TOKEN = "your_api_token_here"

# First pass to create tables with basic columns
node .\create_nocodb_from_schema.js > basic_columns.log 2>&1
# Then open NocoDB and you should see all your tables created with the primitive columns wired up.
# Alternatively, create SQL using the generate_sql_from_schema.py and import this into a fresh NocoDB base.
python generate_sql_from_schema.py --dialect postgres --output schema.sql

# Second pass to create columns that were skipped previously 
node .\create_nocodb_remaining_columns.js > remaining_columns.log 2>&1

# Alternate second pass to create relationships based on linked-record metadata in _schema.json.
node .\create_nocodb_relations_and_rollups.js > relations_rollups.log 2>&1
