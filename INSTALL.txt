Airtable runs on a proprietary cloud and does not need to be installed.

For a NocoDB installation:

On Windows (Quick Test Install) - 

Download and Install using the latest (0.260.1 with support for V2 API) Windows installer executable from https://github.com/nocodb/nocodb.

Access NocoDB: Open your web browser and go to http://localhost:8080/dashboard

Create a new base, MushroomProcess, and determine the BaseID
Create a new token from your profile's Account Settings.

Configure your environment

$env:NOCODB_URL = "http://localhost:8080"
$env:NOCODB_BASE_ID = "pfqzirdzyykqjsr"
$env:NOCODB_API_TOKEN = "WhpknS27GxvON_NiEEVSjm311wyo9NA2CNoNklNK"

# Support intended for v2 or v3 API
$env:NOCODB_API_VERSION = "v2"

# Support intended for recreating links, although this is not currently working
# Link targets are documented instead
$env:NOCODB_RECREATE_LINKS = "false"
$env:NOCODB_RECREATE_ROLLUPS = "false"


Proceed with schema\INSTALL_nocodb.txt
