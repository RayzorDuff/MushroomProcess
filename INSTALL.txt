Airtable runs on a proprietary cloud and does not need to be installed.

For a NocoDB installation:

On Windows (Quick Test Install) - 

Download and Install using the latest (0.263.4 or earlier with support for V2 API) Windows installer executable from https://github.com/nocodb/nocodb.

Access NocoDB: Open your web browser and go to http://localhost:8080/dashboard

Create a new base, MushroomProcess, and determine the BaseID
Create a new token from your profile's Account Settings.

Configure your environment

$env:NOCODB_URL = "http://localhost:8080"
$env:NOCODB_BASE_ID = "pf7ja4bl3alukow"
$env:NOCODB_API_TOKEN = "XmNqqcf7VG1m-yQRpHAS7ZnErLuooT95Y2kYfwNI"

Proceed with schema\INSTALL_nocodb.txt
